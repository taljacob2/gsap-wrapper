# GsapWrapper
This class bundles `gsap` methods together for fast use.

> *NOTE:* the `selector` may be an `#id`, `.class`, `tag` as you wish.
All functions return `Tween` / `gsap.timeline()`.

#### Supported:
- [NPM](/ts)
- [Native JS](/js)

---

> [Guide](https://greensock.com/3-migration/) for gsap 3.
