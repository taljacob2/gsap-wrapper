# GsapWrapper
This class bundles `gsap` methods together for fast use.

> *NOTE:* The `selector` may be an `#id`, `.class`, `tag` as you wish.
All functions return `GsapWrapper`, thus able to be chained.

#### Supported:
- [NPM](/ts)
- [Native JS](/js)
